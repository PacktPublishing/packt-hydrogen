Future Features

What's Next?

This demo is strictly an MVP that demonstrates courses-in-a-box. The assumption is that people will use this on machines without an internet connection, so everything has been built around it with that in mind.

If we choose to go down a route of 'well let's just assume they can be connected to the internet' then the scope changes somewhat, and the need for a single specific application for each course is somewhat diminished. Instead, you could effectively have a 'PacktLib' application that checks for user authentication, grabs associated content, and works a lot more like Steam (which incidentally, is also build in a similar way, a web application in a native container that gives it access to things like the local filesystem).

Most of the changes we can make to offline-bundles are now mostly cosmetic or related to content additions like quizzes and interactive assessments. We've had both working in demos in the past, but due to their nature they're harder to completely test end-to-end without a bit more work. That is likely to be 'version two', once we also have some supporting content to play with.
